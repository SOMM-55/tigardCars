## ۱. خلاصه پروژه
طراحی یک صفحه HTML پیش‌فاکتور (Invoice/Proposal) حرفه‌ای و زیبا که یک برنامه‌نویس full-stack برای کارفرمای خود ارسال می‌کند. این فاکتور شامل بخش‌های مختلف یک سایت معرفی و فروش خودرو است و قیمت هر بخش به صورت مجزا نمایش داده می‌شود. کاربر می‌تواند آیتم‌ها را انتخاب/لغو کند و قیمت نهایی به صورت پویا تغییر کند. همچنین قابلیت چاپ (Print) با استایل اختصاصی دارد.

## ۲. سبک بصری
- **تم:** Modern Dark
- **رنگ‌ها:**
  - پس‌زمینه: `#0f2027` تا `#203a43` (گرادینت تیره)
  - رنگ اصلی (اکسنت): `#00d2ff` (فیروزه‌ای روشن)
  - رنگ ثانویه: `#3a7bd5` (آبی)
  - متن سفید: `#ffffff`
  - متن فرعی: `#8899aa`
  - قیمت: `#00d2ff`
  - تخفیف: `#ff6b6b` (قرمز)
- **زبان:** فارسی
- **واحد پول:** تومان

## ۴. قابلیت‌های فنی
- **HTML Single Page** — تمام کد در یک فایل
- **CSS مدرن** — Flexbox, Grid, Gradient, Glassmorphism, Animations
- **JavaScript پویا** — انتخاب/لغو آیتم‌ها و محاسبه خودکار قیمت
- **Print Stylesheet** — استایل اختصاصی برای چاپ (حذف دکمه‌ها، پس‌زمینه سفید)
- **Responsive** — سازگاری با موبایل و دسکتاپ
- **RTL** — راست‌چین برای زبان فارسی
- **محاسبه مبلغ به حروف** — تابع JavaScript برای نمایش مبلغ به حروف فارسی

## ۵. نیازمندی‌های غیرکاربردی
- ظاهر حرفه‌ای و premium
- سرعت بارگذاری بالا (فایل تکی)
- چاپ تمیز بدون المان‌های اضافی
- کاملاً واکنش‌گرا

## ۶. مراحل پیاده‌سازی
1. نوشتن ساختار HTML (هدر، آیتم‌ها، خلاصه قیمت، فوتر)
2. اضافه کردن CSS استایل (تم Modern Dark)
3. اضافه کردن JavaScript (انتخاب آیتم، محاسبه قیمت، مبلغ به حروف)
4. اضافه کردن Print Stylesheet
5. تست نهایی و تحویل
